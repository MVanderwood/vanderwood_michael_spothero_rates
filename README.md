# VanderwoodMichaelSpotheroRates

## Instructions

1. Unpack file
2. Run application:
  ```
    ruby spothero_rates <json_lot_data_file> <start_time> <end_time>
  ```
3. Run tests:
  ```
    rspec spec/run_specs
  ```

## Dependencies
- [RSpec](https://github.com/rspec/rspec)
- Installation: `gem install rspec`
